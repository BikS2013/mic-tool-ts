# mic-tool

A TypeScript CLI that captures microphone audio on macOS, streams it to the Soniox real-time speech-to-text API via the `@soniox/node` v2 SDK, and renders partial and final transcripts to standard output.

Status: in development. Full usage instructions, prerequisites, configuration reference, and troubleshooting will be filled in during Phase 8 of `docs/design/plan-001-soniox-mic-cli.md`.
